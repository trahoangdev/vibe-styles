# Vibe Styles / Design System Explorer

A powerful tool for exploring, previewing, and customizing popular design systems.

## Features

- **Design System Library**: Explore predefined styles inspired by Apple, Spotify, Linear, and more.
- **Live Preview**: See how components like Dashboards, Auth forms, and Bento grids look with different styles.
- **Theme Editor**: Customize colors, typography, and border radius in real-time.
- **Export**: Generate CSS variables or Tailwind config for your chosen design system.
- **Responsive Preview**: Test your designs on mobile, tablet, and desktop views.
- **Keyboard Shortcuts**: Power-user tools for faster navigation and editing.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   cd vibe-styles
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies Used

- **Framework**: [React](https://reactjs.org/)

- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
